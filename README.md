# IoT Gas Leakage Detection System

## 📌 Project Overview
An IoT-based gas leakage detection system using Arduino, MQ gas sensor, and GSM module. The system monitors gas levels in real time and sends SMS alerts when leakage is detected.

## 🛠 Components Used
- Arduino UNO
- MQ Gas Sensor (MQ-2 / MQ-135)
- GSM Module (SIM800L)
- Buzzer

## 🚀 Features
- Real-time gas detection
- Automatic SMS alert
- Buzzer alarm system
- Low-cost safety solution

## 🔮 Future Enhancements
- Cloud integration
- Mobile application alerts
- Data logging dashboard
